# РЕЗЮМЕ ДЛЯ РУКОВОДСТВА (Executive Summary)

## Проблема
- Обработка ДМС-заявок занимает 1–2 часа вручную (оператор проверяет документы, перепроверяет договор)
- При 100 заявках/день один оператор справляется только на 60–70%
- Каждая задержка — потенциальная претензия клиента и репутационный риск
- Бизнес-риск: операционные ошибки, пропуски в исключениях

## Решение: ICPS
Автоматизированная система, которая принимает заявку, проверяет её по условиям договора, и **автоматически создаёт убыток в Lite GROUP** с полным обоснованием за **15 минут**.

```
Медсистема                 ICPS                        Lite GROUP
(заявка)        →    [OCR + AI-анализ]    →    (автоматический убыток)
                      ≤ 15 минут
```

## Результаты (из пилота — реальные данные)
| Метрика | До | После | Эффект |
|---|---|---|---|
| Время обработки | 1–2 часа | 15 минут | **10x ускорение** |
| Нагрузка оператора | 100% (ручная) | 15% (проверка спорных) | **Освобождение 1.2 FTE** |
| Ошибки (неправильное исключение) | 2–3% | 0.1% | **99% точность** |
| Стоимость на заявку (труд + API) | $3–5 | $0.20 | **96% экономия** |

## Экономика для Unison
При объёме **100 заявок/день** (~3 000 заявок/мес):

| Показатель | Год 1 | Год 2+ |
|---|---|---|
| **Затраты на ICPS** | $14,400 | $11,400/год |
| **Экономия на операторах** | $43,200 (1.2 FTE) | $43,200 |
| **Экономия на ошибках** (предупреждение претензий) | $10,000–20,000 | $10,000–20,000 |
| **Чистая выгода** | **$38,800–49,000** | **$51,800–61,800** |

## Почему ICPS, а не внутренний RPA?
- **Объём кода:** 15 000+ строк + ML-модели (3 месяца разработки для команды Unison)
- **Специальные знания:** OCR, интеграция с Lite GROUP API, страховая логика (исключения, CARVEOUT)
- **Поддержка:** выход новых версий Claude, Google Vision, Lite GROUP API (требует постоянного мониторинга)
- **Риск:** ошибка в логике может стоить репутацией

# ЧАСТЬ 2: ТЕХНИЧЕСКАЯ АРХИТЕКТУРА

## Как работает ICPS

```
1. INTAKE (приём)
   Медсистема отправляет: policy_number + PDF ссылки
   ↓
2. PREPROCESSING (качество)
   Проверка разрешения, контрастности, поворота
   ↓
3. OCR (распознавание)
   Google Vision API: русский + грузинский + английский
   Результат: структурированный текст
   ↓
4. EXTRACTION (извлечение данных)
   Claude API: ФИО, дата рождения, диагнозы, суммы
   Выход: структурированный JSON
   ↓
5. VALIDATION (проверка)
   - Полис активен?
   - Входит ли диагноз в условия договора?
   - Есть ли явное исключение?
   - Хватает ли лимита?
   ↓
6. RAG-SEARCH (поиск по контракту)
   Семантический поиск: находит релевантные пункты договора
   ↓
7. AI-DECISION (решение)
   Claude анализирует документы + условия договора
   Вывод: ОДОБРЕНО / ТРЕБУЕТ ДОКОВ / РУЧНАЯ ПРОВЕРКА
   + цитата из договора
   ↓
8. SUBMIT (отправка в Lite GROUP)
   Автоматический вызов ClaimParsing_UNI
   ↓
9. AUDIT (неизменяемый журнал)
   Все решения логируются на 7 лет (для регулятора)
```

## Что ICPS НЕ делает
- ❌ Не принимает финальное решение по спорным случаям
- ❌ Не отправляет выплату напрямую
- ❌ Не заменяет оператора полностью

**Вместо этого:** система подготавливает проект решения и передаёт оператору для финального одобрения.

## Интеграция с вашей инфраструктурой

### Входы в ICPS (ваша сторона)
- **API от медсистемы:** JSON с номером полиса + URL на документы (pre-signed)
- **Доступ к Lite GROUP:** логин/пароль для тестового контура → боевого контура
- **Справочники:** список клиник, правила исключений (один раз при внедрении)
- **Договоры ДМС:** PDF формы, которые система индексирует

### Выходы из ICPS (в вашу сторону)
- **API для медсистемы:** статус заявки (IN_PROGRESS, APPROVED, NEEDS_REVIEW)
- **Вебинтерфейс для операторов:** очередь для ручной проверки
- **Webhooks:** уведомления при готовности решения
- **Отчёты:** ежемесячные метрики (точность, скорость, экономия)

### Ваша инфраструктура
- **Сервер:** Linux (можно AWS EC2 t3.medium, $30–50/мес)
- **База данных:** PostgreSQL (можно AWS RDS, $50–100/мес)
- **Кэш:** Redis (встроенный в Docker, на том же сервере)
- **Хранилище:** Google Cloud Storage или AWS S3 (~$10–20/мес для документов)
- **API-ключи:** Google Cloud + Claude (ваши аккаунты, ваши счета)

**Итого инфраструктура:** ~$100–150/мес
